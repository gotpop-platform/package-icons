# Package icons

This is a icon package.

## Installation

You can install this package using npm:

```
bun add @gotpop-platform/package-icons
```

## Usage

To use the icon functions, import them in your project:

```typescript
import { Icon } from '@gotpop-platform/package-icons';
```

### Example

```typescript
<Icon />
```
